# Minesweeper Engine

Decision-making portion of the game

## Getting Started

### Pre-requisite(s)

1. Go 1.16+ installed (may work with previous versions as well however it is not tested)

### Build from Source

```text
$ cd engine
$ go build -o ./ ./...
$ ./cli
```
